Practice Problem #31: math combo

Build a function, cocococombomath, that takes 5 numbers as parameters.

It:

  • multiplies the 1st number by the 2nd number
  • to the result of that, it adds the 3rd number
  • to the result of that, it divides by the 4th number
  • to the result of that, it subtracts the 5th number
  • it returns that result

Example:

cocococombomath(3,2,6,3,5)
Returns -1
